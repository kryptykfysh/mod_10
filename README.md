# Mod10

A simple gem to generate mod_10 check digits and check if integers are mod10
valid.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'mod_10'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install mod_10

## Usage

Just use the damn thing.
